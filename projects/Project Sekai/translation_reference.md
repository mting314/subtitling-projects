# Project Sekai AfterTalk Translation Reference

For detailed character and franchise info, see [Sekaipedia](http://sekaipedia.org/wiki/).

## Program Overview

**ProSeka AfterTalk** (プロセカアフタートーク) is a recurring livestream series for the mobile rhythm game Project Sekai: Colorful Stage! feat. Hatsune Miku. Each edition covers one in-game event story: a single voice actor hosts, reads fan letters, and discusses the story while watching key scenes live. Streams air on YouTube after each event ends.

## Franchise Context

- **Game**: Project Sekai: Colorful Stage! feat. Hatsune Miku (プロジェクトセカイ カラフルステージ！ feat. 初音ミク)
- **Short name**: ProSeka (プロセカ), capital S
- **Units**: Five groups of original characters who interact with Virtual Singers (Miku, etc.)
- **School**: Miyamasuzaka Girls Academy (宮益坂女子学園) — Leo/need and MMJ characters attend here
- **SEKAI**: Mysterious worlds born from characters' feelings, each tied to a unit. Refer to these as "SEKAI" (all caps) when discussing the in-game concept
- **AfterTalk**: One word, capital T. Not "After Talk" or "aftertalk"
- **AfterLive**: One word, capital L. Plural is "AfterLives"
- **Event stories**: Referred to as "Episodes", not "Chapters"

## Fixed Translations

Recurring scripted lines that appear across most AfterTalk editions. Use these established translations when the Japanese text matches (allow for minor transcription variations).

### Fan Letter Reading

| Japanese | English | Notes |
|----------|---------|-------|
| おたより | letter / message | Fan submissions. "Letter" when reading one, "messages" in plural/general |
| ラジオネーム / ペンネーム | From: {name}-san | Before reading sender's pen name |
| メッセージありがとうございます | Thanks for the message! | After reading a letter |
