# Project Sekai AfterTalk Translation Reference

For detailed character and franchise info, see [Sekaipedia](http://sekaipedia.org/wiki/).

## Program Overview

**ProSeka AfterTalk** (プロセカアフタートーク) is a recurring livestream series for the mobile rhythm game Project Sekai: Colorful Stage! feat. Hatsune Miku. Each edition covers one in-game event story: a single voice actor hosts, reads fan letters, and discusses the story while watching key scenes live. Streams air on YouTube after each event ends.

## Franchise Context

- **Game**: Project Sekai: Colorful Stage! feat. Hatsune Miku (プロジェクトセカイ カラフルステージ！ feat. 初音ミク)
- **Short name**: ProSeka (プロセカ), capital S
- **Units**: Five groups of original characters who interact with Virtual Singers (Miku, etc.)
- **School**: Miyamasuzaka Girls Academy (宮益坂女子学園) — Leo/need and MMJ characters attend here
- **SEKAI**: Mysterious worlds born from characters' feelings, each tied to a unit. Refer to these as "SEKAI" (all caps) when discussing the in-game concept
- **AfterTalk**: One word, capital T. Not "After Talk" or "aftertalk"
- **AfterLive**: One word, capital L. Plural is "AfterLives"
- **Event stories**: Referred to as "Episodes", not "Chapters"

## Fixed Translations

Recurring scripted lines that appear across most AfterTalk editions. Use these established translations when the Japanese text matches (allow for minor transcription variations).

### Opening Sequence

| Japanese | English | Notes |
|----------|---------|-------|
| プロセカアフタートーク「{イベント名}」編 | Welcome to ProSeka AfterTalk "{Event Name}" Edition! | Title card / opening line |
| 皆さんこんばんは | Good evening, everyone! | Standard greeting |
| {ユニット名}で{キャラ名}役の{声優名}です | This is {VA Name}, voice of {Character Name} from {Unit}. | Self-introduction |
| コンニード | {\i1}Kon-needo{\i0} | Leo/need fan greeting (portmanteau of こんにちは + ニード). Always italicized |
| コンモア | Kon-more | MMJ fan greeting. Always italicized when used as a standalone greeting |

### Program Description

| Japanese | English | Notes |
|----------|---------|-------|
| このプロセカアフタートークは皆さんから頂いたおたよりを紹介しながらイベントを振り返る配信となっております | In this edition of ProSeka AfterTalk, we'll look back at this event while covering messages you, the fans, have sent in. | Standard program description (minor wording varies) |
| 今回は「{イベント名}」編となっています | This time, we'll be covering the "{Event Name}" event. | Event identification |
| 皆様ストーリーはお楽しみいただけましたでしょうか | Did you enjoy the story, everyone? | After event intro |
| 事前にプロセカ公式Discordとメールでおたよりを募集しました | We've gathered your comments for this AfterTalk sent through email and the submissions channel on Discord. | Submission source |
| たくさんのメッセージありがとうございます | Thanks for all the messages, everyone. | After mentioning submissions |

### Fan Letter Reading

| Japanese | English | Notes |
|----------|---------|-------|
| おたより | letter / message | Fan submissions. "Letter" when reading one, "messages" in plural/general |
| ラジオネーム / ペンネーム | From: {name}-san | Before reading sender's pen name |
| メッセージありがとうございます | Thanks for the message! | After reading a letter |

| Japanese | English | Notes |
|----------|---------|-------|

## Character Relationships

Key relationships that may come up in AfterTalk discussions:

- **Shiho** and **Shizuku** Hinomori are sisters (Shiho is Leo/need, Shizuku is MMJ)
- **Akito** and **Ena** Shinonome are siblings (Akito is VBS, Ena is N25)
- **Tsukasa** and **Saki** Tenma are siblings (Tsukasa is WxS, Saki is Leo/need)
- **Mizuki** uses "boku" (ボク) — use "they/them" pronouns when the speaker's intent is ambiguous
