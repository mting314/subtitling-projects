# Lieraji Translation Reference

## Program Overview

**Lieraji** (リエラジ) = "Liella no Radio Japan" (リエラのRadio Japan), a weekly radio show for the Love Live! Superstar!! franchise. Features voice actors from Liella! discussing the franchise, reading listener mail, and playing games.

## Franchise Context

- **Series**: Love Live! Superstar!!
- **Group**: Liella!
- **School**: Yuigaoka Girls' High School (結ヶ丘女子高等学校)
- **Format**: Weekly radio show, ~30 minutes per episode
